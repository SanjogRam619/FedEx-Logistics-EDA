# 📦 FedEx Logistics Supply Chain Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the SCMS Delivery History Dataset to uncover insights related to shipment operations, freight costs, vendor performance, and delivery efficiency.

---

## 🎯 Business Objective

- Analyze shipment trends
- Optimize freight cost
- Evaluate vendor performance
- Improve delivery efficiency
- Support business decision-making

---

## 🛠 Tools & Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

---

## 📊 Project Workflow

- Data Loading
- Data Cleaning
- Data Wrangling
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis
- Business Recommendations

---

## 📈 Key Insights

- Air is the most used shipment mode.
- Nigeria contributes the highest line item value.
- ARV is the highest-value product group.
- SCMS from RDC is the leading vendor.
- Freight cost is positively related to shipment value.
